# Future Ideas

> Non-committal ideas for phases beyond the roadmap. Not planned, not promised.

---

## Gameplay Extensions

- **Procedural dungeons**: AI-generated stage layouts based on a topic keyword
- **Skill trees**: unlock passive bonuses (hint cooldown reduction, XP multipliers)
- **PvP challenge mode**: two players race to complete the same challenge
- **Speedrun leaderboards**: fastest time to complete a world (per stage time tracking)
- **Prestige system**: wipe XP/progress at max level for a rare cosmetic reward
- **Daily dungeon**: time-limited stage available for 24 hours, unique rewards

---

## AI Extensions

- **AI dungeon master**: generates live narrative during stage based on player choices
- **Voice-guided learning**: AI narrates concept chunks (text-to-speech)
- **Adaptive curriculum**: AI builds a custom world order based on learning gaps
- **Code review as boss fight**: AI reviews submitted code and generates boss health attacks based on quality
- **Conversational NPC**: in-world NPCs powered by Claude — ask any question about the concept

---

## Social Extensions

- **Guild wars**: competing guilds complete the same world on a timer; highest collective XP wins
- **Mentor system**: experienced players tutor new players on their weak stages
- **World sharing**: share a stage link; friend plays from same starting state
- **Replay theater**: watch a high-scorer's playthrough of a stage

---

## Content Extensions

- **Video concept chunks**: short animated video explanations alongside text
- **Interactive code playground**: embedded REPL in concept phase (Sandpack)
- **Diagram quest**: drag-and-drop diagram completion (e.g., fill in a flowchart)
- **Real-world project quests**: multi-stage arc where player builds a mini project
- **Topic submission**: community votes on next world topics

---

## Business Extensions

- **B2B bootcamp mode**: fork ForgeMinds for coding bootcamp curriculum delivery
- **API-as-product**: let third-party apps embed ForgeMinds worlds as learning widgets
- **Certification**: completion certificates (PDF + shareable badge) for world mastery
- **Custom AI tutor**: enterprise customers bring their own AI model/persona
- **Analytics SDK**: organizations track employee learning progress via webhook events

---

## Technical Extensions

- **Offline mode**: Service Worker + IndexedDB for fully offline stage completion
- **Native mobile**: React Native wrapper using the same game engine abstraction
- **WebGL shaders**: custom GLSL shaders for atmospheric world effects in PixiJS
- **WebAssembly code evaluator**: run player-submitted code in a WASM sandbox for security
- **Collaborative real-time editing**: content creator team edits world JSON simultaneously (CRDTs)
